Usando a variável Const.

O operador CONST tem o comportamento similar ao LET, porém estaremos criando uma constante, que não irá sofrer alterações ao longo do uso no script.

Obs: é uma boa prática as constantes serem criadas com caracteres maiúsculos. No momento de sua criação ele também já deve receber um valor para não dar erro.

Muito importante: geralmente guardamos senhas de banco de dados, links para APIs e parâmetros de configuração para nossa aplicação dentro de variáveis constantes, quando recuperamos estas variáveis temos a segurança qua não haverá alterações porque o próprio Javascript não vai permitir.
