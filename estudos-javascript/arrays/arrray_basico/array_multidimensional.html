Array multidimensional

Obs: basicamente podemos colocar dentro dos imdices dos nossos, outros arrays de dimensões necessárias para atender nossas necessidades.

Obs: podemos ter dois arrays isolados dentro da aplicação ou arrays multidimensionais, indices apontando para outros arrays.

Este tipo de estrutura facilita a leitura de dados em sistemas mais complexos.

O atributo length contabiliza corretamente apenas se os indices forem numéricos, partirem da posiçãp zero e forem sequenciais.
