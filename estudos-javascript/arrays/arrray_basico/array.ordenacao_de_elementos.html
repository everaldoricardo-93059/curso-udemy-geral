Ordenação de elementos por ordem alfabética em Arrays

Método sort() - Faz a ordenação por ordem alfanumérica



Quando trabalhamos com arrays essencialmente numéricos precisamos acrescentar uma função para ensinar o método sort() a fazer a ordenação correta baseada em valores.



Se todos os valores contidos no array forem valore numéricos mesmo em formato de string, o Javascript reconhecerá e fará a ordenação correta, porém, se apenas alguns elementos forem strings a ordenação será incorreta.
