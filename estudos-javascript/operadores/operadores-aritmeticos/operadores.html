Operadores Aritméticos

Obs: O incremento pré(esquerda da variável) vai atuar dentro da instrução que a variável está inserida, o incremento pós(direita da variável) apenas após a instrução ser atualizada.

Obs: Os valores recuperados dentro de um sistema, seja com o prompt ou http, ou outro modo qualquer, seja banco de dados, vão sempre vir em formato de string. Por isso, muito cuidado com o tratamento de dados dentro do programa, principalmente com operações de soma que podem viram concatenação, tanto a função parseInt para transformar string em valor numérico deve ser usada quanto também a função toString() para realizar o caminho inverso.
