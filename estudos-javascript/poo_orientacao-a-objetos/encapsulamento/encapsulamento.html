Encapsulamento - POO

O encapsulamento consiste em encapsular um objeto com o objetivo de deixá-lo seguro. O objeto vai ter condições de dizer o que dentro dele está disponível pro sistema. Nós podemos assim, determinar, dentro do objeto, que atributo ou método, é publico ou privado, criando assim uma camada de segurança. Como o operador private ainda não está disponível para uso nas aplicações, temos que usar o (_) para indicar que aquele item é privado.

Para acessar atributos privados do objeto usamos os métodos getters e setters.

Os getters acessam os dados privados e os setters os modificam.

Embora no Javascript se possa recuperar a referência para o objeto e alterá-lo, a convenção em termos de encapsulamento orienta que utilizemos o método set para faze-lo, isto por medida de segurança.
