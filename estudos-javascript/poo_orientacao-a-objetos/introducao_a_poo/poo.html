Orientação a objetos

O que é paradigma??

Padrão, conduta. Forma de se fazer alguma coisa

Obs: o paradigma de orientação a objetos não é o sucessor do paradigma procedural porque na prática eles se completam. O Javascript suporta os dois paradigmas.


O que é programação reativa??

A programação reativa é uma boa opção para aplicativos que precisam lidar com grandes volumes de dados em tempo real, garantir alta performance e escalabilidade, e fornecer respostas rápidas e confiáveis aos usuários
