Abstração

Abstração é muito mais uma qualidade intuitiva, de conhecimento das coisas do que de programação.

Temas que fazem parte do conhecimento da abstrção

  1. Entidade
  2. Identidade
  3. Características(Atributos)
  4. Ações(Métodos)

Obs: quando estamos definindo classes, é uma boa prática utilizar a instrução CamelCase, ou seja, letras maiúsculas no início das palavras. Na declaração das características devenos usar o CamelCase apenas na segunda palavra.

Obs: o constructor() é o método automatico para instanciar a entidade, em outras palavras, construir o objeto, é onde se declara os atríbutos do objeto. Neste local utilizaremos o operador this, ele tem a inteligência para conectar os atríbutos ao contexto do objeto, ou seja, uma varável que estará encapsulada dentro do objeto.

Como declaramos as acões ou métodos?

As acões são definidas com a mesma sintaxe de funções, porém quando declaradas dentro de um modelo são definidas como métodos. Neste caso não estamos declarando a palavra reservada function porque estamos usando o Es6, esse tipo de contrução de classe é uma festure desta versão, se estivessemos usando uma versão anterior precisaríamos do tradutor BABEL.

Agora para criar a entidade temos que dar a ela uma identidade. O new é o operador que indica para o interpretador do Javascript que eu quero transformar um modelo em um objeto, a partir desta instancia nós podemos recuperar os atributos e métodos do objeto.
