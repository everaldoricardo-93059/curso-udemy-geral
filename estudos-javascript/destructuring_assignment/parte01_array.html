Destructuring Assignment

Função do operador Destructuring: ele é um operador de desestruturação. Basicamete falando, ele tira valores de dentro de uma estrutura, que pode ser um Array ou um Objeto.


Nós usamos o operador Destructuring basicamente para evitar a verbosidade do código. Temos que indicar qual é o operador que usaremos para declarar a variável
