Funções Factory

Através do padrão de projeto Factory nós podemos implementar uma "Fábrica" para criação dos nossos objetos em tempo de execução. Basicamente falando, em Javascript, as funções Factory combinam a chamada de funções com o retorno de objetos literais, que podem ser criados com base em parâmetros que são passados pra essas respectivas funções.

Funções Factory servem para criação de objetos de forma dinâmica, ou em tempo de execução, porque basta chamar a função, passando ou não parâmetros, para termos o retorno de um objeto.

Obs: se essa informações fossem originadas de um banco de dados, nós teríamos que fazer uma requisição HTTP para algum servidor, esse servidor iria processar essa requisição e na sequência daria uma resposta HTTP para nós com um JASON anexado a resposta, que seria convertido dentro da aplicação também por um objeto literal, e a partir dai iríamos customizando estes parâmetros, mas isso seria impraticavel, por isso as funções Factory são utilizadas.
