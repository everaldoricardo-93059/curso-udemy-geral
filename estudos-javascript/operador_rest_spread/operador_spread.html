Operadores Rest(juntar estrutura de dados) e Spread(espalhar estruturas de dados)

O operador em si é contituído por 3(...) pontos, o que define se ele vai ser Rest ou Spread é o contexto onde ele está sendo utilizado.

Contexto Spread(Observe no console!)

O operador Spread serve para espalhar um Array dentro de outra estrutura. Podemos posicionar os elementos no inicio ou no final.
