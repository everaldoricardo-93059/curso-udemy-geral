Selecionando elementos no DOM


Principais métodos de acesso a elementos da página

  1. getElementById()- seleciona através do id. Devemos ter apenas um id por página. ele é usado em 85% dos casos.

  2. getElementsByTagName() - é possível a seleção de varios elementos com mesmo identificador()tag
  3. getElementdByClassName - retorna elementos com a mesma classe

  4. getElementsByName - usado em elementos radio ou checkbox com varios names iguais, seu retorno é um node list
