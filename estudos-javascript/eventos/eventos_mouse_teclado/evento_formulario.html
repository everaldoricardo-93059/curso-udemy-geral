Eventos javascrip de formulário

Principais eventos

  1. onfocus - quando o elemento recebe o foco do cursor do mouse

  2. onblur - quando o elemento perde o foco

  3. onchange - quando o estado do elemento é modificado

    4. Obs: este elemento é muito utilizado com o elemento select


    Obs: usamos o console.log para teste porque se usassemos o alert perderiamos o foco no momento do seu dispara e teriamos que dar ok para retorno do foco
