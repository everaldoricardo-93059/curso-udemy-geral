Eventos no site

Eventos são interações do usuário no uso da página do site(Ex: clique do mouse, apertar uma tecla no teclado, etc...

Principais eventos capturados pelo javascript através do uso do mouse

  1. Onclick - quando o elemento é clicado.

  2. ondblclick -quando o elemento é clicado duas vezes

  3. onmouseup - quando um click sob determinado elemento é liberado

  4. onmouseover - quando o cursor do mouse sobrepôe um elemento

  5. onmouseout - quando o cursor do mouse sai da região da página ocupada pelo elemento
