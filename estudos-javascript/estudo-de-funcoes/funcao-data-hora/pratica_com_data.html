Prática com o uso do método Date

Adidionar e remover dias

No momento da instanciação do objeto Date podemos setá-lo para efetuar cálculos entre duas datas.



Obs: o método getTime recupera os milisegundos entre duas datas para fazermos cálculos entre elas, usando como base a data de "01 de janeiro de 1970"(Padrao computacional de sistemas operacionais baseados no UNIX).
