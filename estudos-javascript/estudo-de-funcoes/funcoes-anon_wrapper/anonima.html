Funções anônimas e a técnica de Wrapper (embrulhar).


Funções anônimas não tem seu nome declarado mas o funcionamento é idêntico as funções normais.

Obs: funções anônimas são ideais para serem usadas como parâmetro de callback para outras funções.


O javascript incorpora um conceito muito forte de callback, que é a passagem de funções como parâmetro para outras funções.
