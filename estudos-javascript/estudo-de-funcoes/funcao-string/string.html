Funções para manipular Strings

String Jorge Sant Ana

Obs: a função length() retorna o número de caracteres da String

As strings sempre iniciam pelo valor 0(zero). Os caracteres vazios também são contados.


Obs: a função Charat() retorna valor da do caracter dentro do indice da cadeia de caracteres que formam a string.


Obs: o método indexOf() retorna o valor da primeira ocorrência de determinado caracter dentro da cadeia de caracteres da string - importante: ele faz diferenciação entre caracteres maiusculos minusculos. Quando não houver o caracter na string ele retorna o valor -1


O método replace() permite substituir textos dentro de uma string


O método substr() permite extrair parte de uma string com base em uma posição inicial seguida de um valor que representa a quantidade de caracteres a partir daquela posição inicial


Outros dois métodos bastante utilizados são o toLowerCase() e o método toUpperCase() que mudam os caracteres para minusculo ou maisculo.


O método trim() remove os espaços nas duas extremidades da string
