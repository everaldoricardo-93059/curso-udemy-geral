Estrutura de repetição Foreach

O Foreach é uma função pertencente ao objeto Array()

Para usar o Foreach precisamos recuperar o Array(Obs: função do prototype do objeto array)

O Foreach aguarda como parâmetro uma função de callback()Obs: uma função que indica ao foreach como ele tem que atuar em cada um dos indices do Array - Valor/Indice/Array(Obs: poderiam ser qualquer nome, o importante é a sequência)

Em javascript os parâmetros das funções são variáveis

Uso do Foreach usando o recurso de parâmetros variáveis

Neste exemplo usaremos apenas o retorno do primeiro parâmetro(valor)

Em javascript os parâmetros das funções são variáveis



Em javascript funções podem ser atribuídas a variáveis

Obs: é interessante encaminhar a função como variável porque se houver outros arrays dentro da aplicação nós podemos encaminha a mesma variável para tratar índices de outros arrays



O forEach atua apenas em valores numéricos a partir de 0, não precisa ser sequencial, mas precisa ser numérico, caso contrário ele desconsidera a posição do array. Neste caso devemos usar o for in
