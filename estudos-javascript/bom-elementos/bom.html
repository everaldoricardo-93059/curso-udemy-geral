Bom - Browser Object Model

Obs: é uma forma de se comunicar através de javascript com recursos do browser

Por exemplo:

1 - Acessar o histórico de navegação

2 - Forçar o redirecionamento de URL

3 - Salvar e recuperar cookies

Essas ações não estão ligadas a elementos HTML mas a recursos do próprio browser

Quando nossos scripts HTML são interpretados pelo browser, nós podemos através do javascript acessar elementos da página através da API DOM quanto recursos do próprio browser com a API BOM, com isso podemos manipular recursos internos e externos da nossa aplicação Web.
